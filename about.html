关于本站

一个面向爱好者的牡丹鹦鹉品种图鉴，帮助认识常见羽色变异。

网站简介

「牡丹鹦鹉图鉴」收录了 6 种常见的桃脸牡丹鹦鹉（Agapornis roseicollis）羽色变异， 包括原始绿、黄化、白脸、白脸蓝、紫罗兰和珍珠。本站旨在帮助新手鸟友认识不同品种的外观特征， 便于选购、交流与学习。

饲养基础知识

🌾

饮食

以优质鹦鹉粮为主，搭配新鲜蔬果。避免巧克力、牛油果、高盐高糖食物。

🏠

笼舍

笼体宜宽敞，配备栖木、玩具。每日保证安全放飞时间，促进运动。

💊

健康

羽色与品种无关健康。关注精神、粪便、羽毛状态，定期清洁，必要时就医。

🤝

互动

牡丹鹦鹉聪明亲人，需每日陪伴。单只饲养应给予足够关注，或考虑成对（需了解配对习性）。

图片文件说明

品种图片存放在 images/ 文件夹，已使用清晰命名：

  • normal-green.jpg — 原始绿
  • lutino.jpg — 黄化
  • white-faced.jpg — 白脸
  • blue.jpg — 白脸蓝
  • violet.jpg — 紫罗兰
  • opaline.png — 珍珠

替换某品种照片时，保持文件名不变即可，网站会自动更新。

上线发布（GitHub Pages）

本站为纯静态网页，可免费发布到 GitHub Pages。项目已初始化 Git，按以下步骤操作：

  1. 注册 GitHub 账号（若还没有）
  2. 在 GitHub 新建仓库，名称如 lovebird-site，选 Public
  3. 在本项目文件夹打开终端，执行： 
    git remote add origin https://github.com/你的用户名/lovebird-site.git
git branch -M main
git push -u origin main
  4. 进入仓库 → Settings → Pages → Source 选 Deploy from a branch
  5. Branch 选 main，文件夹选 / (root)，保存
  6. 等待 1～2 分钟，访问 https://你的用户名.github.io/lovebird-site/
